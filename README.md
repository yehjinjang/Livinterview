# ZIPUP

## 📌 Overview
ZIPUP은 사용자의 라이프스타일에 맞춘 맞춤형 LLM 레포트를 제공하는 프로젝트
개인화된 인사이트와 데이터 기반 리포트를 통해 보다 스마트한 일상 관리가 가능하도록 지원합니다.

## ⏳ **Period**  
2025년 4월 10일 ~ 2025년 5월 26일
